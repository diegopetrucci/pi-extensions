# pi-extensions

A collection of pi extensions.

## Included extensions

- [`minimal-footer`](./extensions/minimal-footer) — replaces pi's built-in footer with a minimal two-line layout.

## Install

Install from GitHub:

```bash
pi install git:github.com/diegopetrucci/pi-extensions
```

Then reload pi:

```text
/reload
```

## Repo structure

Each extension lives in its own folder and can also have its own `package.json` metadata for standalone publishing later.
