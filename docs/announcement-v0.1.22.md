# Announcement copy — v0.1.22

## Short post

Released `@diegopetrucci/pi-extensions@0.1.22`.

This release updates `openai-fast`: running `/fast` without arguments now opens a picker for status/on/off/auto/toggle, while explicit commands like `/fast on` and `/fast status` continue to work unchanged.

Install:

```bash
pi install npm:@diegopetrucci/pi-extensions
```

or just openai-fast:

```bash
pi install npm:@diegopetrucci/pi-openai-fast
```
