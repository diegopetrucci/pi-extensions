# Announcement copy — v0.1.23

## Short post

Released `@diegopetrucci/pi-extensions@0.1.23`.

This release updates `librarian`: local repo checkout caching is now enabled by default, and `/librarian-cache on|off|toggle|status` controls it across interactive, RPC, print, and JSON runs.

Install:

```bash
pi install npm:@diegopetrucci/pi-extensions
```

or just librarian:

```bash
pi install npm:@diegopetrucci/pi-librarian
```

## Discord / forum version

`pi-extensions@0.1.23` is out.

This release updates `librarian`: local GitHub repo checkout caching is now enabled by default. Use `/librarian-cache on`, `/librarian-cache off`, `/librarian-cache toggle`, or `/librarian-cache status` to control it. The setting persists globally and works in interactive, RPC, print, and JSON modes.

Full collection:

```bash
pi install npm:@diegopetrucci/pi-extensions
```

Standalone librarian:

```bash
pi install npm:@diegopetrucci/pi-librarian
```
