# Announcement copy — v0.1.9

## Short post

Released `@diegopetrucci/pi-extensions@0.1.9`.

This upgrades the `notify` extension for pi with configurable terminal, desktop, bell, and sound notifications.

It also adds merged config file support and a sample config file.

Install:

```bash
pi install npm:@diegopetrucci/pi-notify
```

or:

```bash
pi install npm:@diegopetrucci/pi-extensions
```

## Discord / forum version

I just shipped `v0.1.9` of my pi extensions package.

This release upgrades the `notify` extension so it can send terminal notifications, desktop notifications, a terminal bell, and sound playback when the agent finishes and is ready for input. All channels are enabled by default, and the extension now supports merged global and project config files plus a ready-to-copy sample config.

Install the standalone package with:

```bash
pi install npm:@diegopetrucci/pi-notify
```

Or install the full collection:

```bash
pi install npm:@diegopetrucci/pi-extensions
```
