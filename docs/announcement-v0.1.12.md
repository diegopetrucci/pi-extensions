# Announcement copy — v0.1.12

## Mastodon / Bluesky / LinkedIn version

Just released `@diegopetrucci/pi-extensions@0.1.12`.

This refreshes the `oracle` extension's model preference matrix for the latest top reasoning models in pi: Claude Opus 4.7, GPT-5.5 Pro / GPT-5.5, Gemini 3.1 Pro, Grok 4.3, Kimi K2.6, DeepSeek V4 Pro, MiniMax M2.7, Mistral Medium 3.5, and more.

It also adds preference coverage for newer built-in providers including Cloudflare, DeepSeek, Fireworks, and Moonshot.

```bash
pi install npm:@diegopetrucci/pi-oracle
```

or the full collection:

```bash
pi install npm:@diegopetrucci/pi-extensions
```
