# Announcement drafts — v0.1.52

## Short

Released `@diegopetrucci/pi-extensions@0.1.52` with a new `contrarian` extension: a read-only adversarial-analysis subagent that stress-tests plans, designs, assumptions, bug hypotheses, and conclusions by steelmanning the strongest opposing case.

Standalone package: `@diegopetrucci/pi-contrarian@0.1.0`.

## Install

```bash
pi install npm:@diegopetrucci/pi-extensions
# or just contrarian
pi install npm:@diegopetrucci/pi-contrarian
```

Then reload pi with `/reload`.
