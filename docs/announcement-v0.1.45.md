# Announcement copy — v0.1.45

## Short post

Released `@diegopetrucci/pi-extensions@0.1.45` and `@diegopetrucci/pi-oracle@0.1.12`.

This release updates `oracle` auto-selection so Claude Fable 5 is preferred when available on Anthropic-compatible providers, with Claude Opus 4.8 still retained as a fallback where exposed.

Install:

```bash
pi install npm:@diegopetrucci/pi-extensions
```

or standalone:

```bash
pi install npm:@diegopetrucci/pi-oracle
```

Standalone package: `@diegopetrucci/pi-oracle@0.1.12`.
