# Announcement copy — v0.1.34

## Short post

Released `@diegopetrucci/pi-extensions@0.1.34` and `@diegopetrucci/pi-librarian@0.1.2`.

This release shortens `librarian`'s default local repository checkout cache window from 30 days to 7 days. Caching is still enabled by default and still controlled with `/librarian-cache on | off | toggle | status`.

Install:

```bash
pi install npm:@diegopetrucci/pi-extensions
```

or standalone:

```bash
pi install npm:@diegopetrucci/pi-librarian
```

Standalone package: `@diegopetrucci/pi-librarian@0.1.2`.
