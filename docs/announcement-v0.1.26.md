# Announcement copy — v0.1.26

## Short post

Released `@diegopetrucci/pi-extensions@0.1.26`.

This release adds `triage-comments`: a read-only review-comment triage extension with `/triage-comments` and a `triage_comments` subagent tool for classifying selected PR feedback with evidence, draft responses, and handling options before implementation.

Install:

```bash
pi install npm:@diegopetrucci/pi-extensions
```

or standalone:

```bash
pi install npm:@diegopetrucci/pi-triage-comments
```

Standalone package: `@diegopetrucci/pi-triage-comments@0.1.0`.

## Discord / forum version

`pi-extensions@0.1.26` is out.

This release adds `triage-comments`, a read-only review-comment triage extension. Use `/triage-comments` to paste reviewer feedback or fetch PR comments, then have the `triage_comments` subagent tool classify selected comments with evidence, suggested responses, and handling options.

Full collection:

```bash
pi install npm:@diegopetrucci/pi-extensions
```

Standalone package:

```bash
pi install npm:@diegopetrucci/pi-triage-comments
```

The standalone package version is `@diegopetrucci/pi-triage-comments@0.1.0`.
