# Announcement copy — v0.1.14

## Short post

Released `@diegopetrucci/pi-extensions@0.1.14`.

This update moves the extensions to Pi's current `@earendil-works/*` package imports and repository links, switches oracle to `typebox`, and refreshes oracle provider preferences for the current Pi catalog.

Install:

```bash
pi install npm:@diegopetrucci/pi-extensions
```

or just oracle:

```bash
pi install npm:@diegopetrucci/pi-oracle
```

## Discord / forum version

`pi-extensions@0.1.14` is out.

This release updates the extension collection for Pi's move to `earendil-works/pi-mono` and the `@earendil-works/*` package scope. It also switches the oracle extension from `@sinclair/typebox` to `typebox`, removes stale Google Gemini CLI / Antigravity provider preferences, and adds hardcoded oracle preferences for the current Xiaomi MiMo providers.

Full collection:

```bash
pi install npm:@diegopetrucci/pi-extensions
```

Standalone oracle:

```bash
pi install npm:@diegopetrucci/pi-oracle
```
