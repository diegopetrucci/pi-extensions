Packages `illustrations-to-explain-things` as an installable pi skill for Xiaohei-style article illustrations and visual metaphors.

## Highlights

- Adds `@diegopetrucci/pi-illustrations-to-explain-things@0.1.0`.
- Registers the skill in the collection package via `pi.skills`.
- Bundles the skill instructions, references, OpenAI skill metadata, and example images.
- Supports article/blog illustrations, shot lists, image edits, and visual metaphors.

## Packages

- `@diegopetrucci/pi-extensions@0.1.44`
- `@diegopetrucci/pi-illustrations-to-explain-things@0.1.0`

## Install

Standalone:

```bash
pi install npm:@diegopetrucci/pi-illustrations-to-explain-things
```

Collection:

```bash
pi install npm:@diegopetrucci/pi-extensions
```

Then reload pi:

```text
/reload
```
