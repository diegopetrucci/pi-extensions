# Announcement drafts — v0.1.55

## Short

Released `@diegopetrucci/pi-extensions@0.1.55` and new standalone `@diegopetrucci/pi-code-reviewer@0.1.0`: adds `code-reviewer`, a read-only `code_reviewer` subagent tool for isolated code reviews focused on ticket fit, diff mismatches, correctness, security, simplicity, and validation gaps, with contrarian-style model selection and thinking-level clamping.

## Packages

- `@diegopetrucci/pi-extensions@0.1.55`
- `@diegopetrucci/pi-code-reviewer@0.1.0`

## Install

```bash
pi install npm:@diegopetrucci/pi-extensions
# or standalone
pi install npm:@diegopetrucci/pi-code-reviewer
```

Then reload pi with `/reload`.
