# Announcement drafts — v0.1.55

## Short

Released `@diegopetrucci/pi-extensions@0.1.55` plus new standalone packages `@diegopetrucci/pi-code-reviewer@0.1.0` and `pi-dynamic-context-pruning@0.1.0`: adds `code-reviewer` for isolated guarded code reviews and `dynamic-context-pruning` for cache-aware cleanup of stale, duplicate, and superseded session context.

## Packages

- `@diegopetrucci/pi-extensions@0.1.55`
- `@diegopetrucci/pi-code-reviewer@0.1.0`
- `pi-dynamic-context-pruning@0.1.0`

## Install

```bash
pi install npm:@diegopetrucci/pi-extensions
# or standalone
pi install npm:@diegopetrucci/pi-code-reviewer
pi install npm:pi-dynamic-context-pruning
```

Then reload pi with `/reload`.
