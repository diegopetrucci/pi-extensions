# Announcement drafts — v0.1.58

## Short

v0.1.58 fixes `pi-minimal-footer` on Pi v0.81.1 by removing the removed `AuthStorage.create()` startup path and using model-registry auth for OpenAI Codex usage, with legacy/stored-auth fallbacks preserved. It also includes Pi 0.81 SDK/catalog alignment, release tooling hardening for npm unpublished-version detection, and package catch-up publishes for the root collection and standalone artifacts.
