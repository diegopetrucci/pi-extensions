# Announcement copy — v0.1.35

## Short post

Released `@diegopetrucci/pi-extensions@0.1.35` and refreshed all standalone extension packages for Pi `0.76.0` compatibility.

This release ships `.pi-fleet-tested-version` markers in the collection and every standalone package, updates the local Pi dev baseline to `0.76.0`, and fixes Oracle for Pi `0.76.0` tool/result typing.

Install:

```bash
pi install npm:@diegopetrucci/pi-extensions
```

or a standalone package, for example:

```bash
pi install npm:@diegopetrucci/pi-oracle
```
