# Announcement copy — v0.1.16

## Short post

Released `@diegopetrucci/pi-extensions@0.1.16`.

This release adds `compact-bash`, a new pi extension that makes collapsed assistant `bash` tool output much quieter: collapsed bash rows show a one-line preview by default, while full output is still available with `Ctrl+O`.

Install:

```bash
pi install npm:@diegopetrucci/pi-extensions
```

or just compact-bash:

```bash
pi install npm:@diegopetrucci/pi-compact-bash
```

## Discord / forum version

`pi-extensions@0.1.16` is out.

This release adds `compact-bash`, a new pi extension that makes collapsed assistant `bash` tool output much quieter. It shows a one-line preview plus a hidden-line count by default, and keeps full rendered output available with `Ctrl+O`.

Commands:

```text
/compact-bash status
/compact-bash off
/compact-bash on
/compact-bash toggle
```

Full collection:

```bash
pi install npm:@diegopetrucci/pi-extensions
```

Standalone compact-bash:

```bash
pi install npm:@diegopetrucci/pi-compact-bash
```
