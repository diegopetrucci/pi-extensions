# Announcement — v0.1.48

Released `@diegopetrucci/pi-extensions@0.1.48` and `@diegopetrucci/pi-minimal-footer@0.1.10`.

This release fixes [#4](https://github.com/diegopetrucci/pi-extensions/issues/4): `minimal-footer` now shows Pi's experimental-features `xp` marker when `PI_EXPERIMENTAL=1`. Prompt cache-hit display remains intentionally omitted from `minimal-footer`.

## Short post

Released `pi-extensions` v0.1.48 with `minimal-footer` support for Pi's experimental `xp` footer marker.

When `PI_EXPERIMENTAL=1`, `minimal-footer` can now show `xp` in the compact bottom-left footer status area.

## Tiny post
