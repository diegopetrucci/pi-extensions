# Announcement copy — v0.1.32

## Short post

Released `@diegopetrucci/pi-extensions@0.1.32` and `@diegopetrucci/pi-review@0.1.0`.

This release adds `review`, a standalone extension adapted from `mitsuhiko/agent-stuff` that provides `/review` and `/end-review` for uncommitted changes, base branches, commits, GitHub PRs, and selected paths. It also carries forward upstream custom review instructions and loop-fixing behavior for compatible targets.

Install:

```bash
pi install npm:@diegopetrucci/pi-extensions
```

or standalone:

```bash
pi install npm:@diegopetrucci/pi-review
```

Standalone package: `@diegopetrucci/pi-review@0.1.0`.

License note: `extensions/review` remains Apache-2.0, so the root collection package metadata is mixed-license (`MIT AND Apache-2.0`).

## Discord / forum version

`pi-review@0.1.0` / `pi-extensions@0.1.32` is out.

This release adds `review`, a standalone tlh/pi extension adapted from `mitsuhiko/agent-stuff`. It provides `/review` and `/end-review` for interactive review flows over uncommitted changes, base branches, specific commits, GitHub pull requests, and selected paths. The extension also keeps upstream support for shared custom review instructions and loop-fixing behavior on compatible targets.

Full collection:

```bash
pi install npm:@diegopetrucci/pi-extensions
```

Standalone package:

```bash
pi install npm:@diegopetrucci/pi-review
```

The standalone package version is `@diegopetrucci/pi-review@0.1.0`.
