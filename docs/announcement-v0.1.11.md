## Discord / forum version

I just shipped `v0.1.11` of my pi extensions package.

This release upgrades `minimal-footer` so it can show OpenAI Codex subscription usage directly in the footer. When the active provider is `openai-codex`, the footer now appends the current `5h` and `7d` usage percentages alongside the usual context usage. Other providers keep the same minimal footer behavior as before.

Install the standalone package with:

```bash
pi install npm:@diegopetrucci/pi-minimal-footer
```

Or install the full collection:

```bash
pi install npm:@diegopetrucci/pi-extensions
```
