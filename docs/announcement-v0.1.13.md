# Announcement copy — v0.1.13

## Short post

Released `@diegopetrucci/pi-extensions@0.1.13`.

This update makes `minimal-footer` configurable and adds an optional red `DUMB ZONE` indicator when context usage goes above 200k tokens. You can also disable or relabel the OpenAI Codex `5h` / `7d` usage windows.

Install:

```bash
pi install npm:@diegopetrucci/pi-minimal-footer
```

or:

```bash
pi install npm:@diegopetrucci/pi-extensions
```

## Discord / forum version

`pi-minimal-footer@0.1.5` / `pi-extensions@0.1.13` is out.

This release makes `minimal-footer` configurable via JSON. It adds an optional red `DUMB ZONE` indicator above 200k context tokens, and lets you disable, relabel, or tune that threshold. The OpenAI Codex `5h` and `7d` usage windows are also configurable and can be disabled independently or entirely.

Standalone install:

```bash
pi install npm:@diegopetrucci/pi-minimal-footer
```

Full collection:

```bash
pi install npm:@diegopetrucci/pi-extensions
```
