v0.1.31 adds `gnosis`, an agent-facing pi tool for the `gn` repo-local knowledge base CLI.

Agents can now search and record durable project decisions, constraints, rejected alternatives, and intent through gnosis without adding user slash commands. The wrapper supports plan/review guidance, search/latest/show/topics, write, and reindex, while intentionally leaving destructive edit/rm operations out of the tool surface.

Packages:
- `@diegopetrucci/pi-extensions@0.1.31`
- `@diegopetrucci/pi-gnosis@0.1.0`
