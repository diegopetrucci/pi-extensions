# Announcement copy — v0.1.8

## Short post

Released `@diegopetrucci/pi-extensions@0.1.8`.

This adds a small `notify` extension for pi that sends a terminal or desktop notification when the agent finishes and is waiting for input.

It is adapted from the original example in `badlogic/pi-mono` and kept basically the same.

Install:

```bash
pi install npm:@diegopetrucci/pi-notify
```

or:

```bash
pi install npm:@diegopetrucci/pi-extensions
```

## Discord / forum version

I just shipped `v0.1.8` of my pi extensions package.

This release adds a small `notify` extension that sends a terminal or desktop notification when the agent finishes and is waiting for input. It supports OSC 777 terminals, Kitty, and Windows toast notifications depending on the current environment.

It is adapted from the original example in `badlogic/pi-mono`, with the implementation kept basically the same.

Install the standalone package with:

```bash
pi install npm:@diegopetrucci/pi-notify
```

Or install the full collection:

```bash
pi install npm:@diegopetrucci/pi-extensions
```
