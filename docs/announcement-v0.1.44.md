Released `@diegopetrucci/pi-extensions@0.1.44` with a new packaged skill:

- `@diegopetrucci/pi-illustrations-to-explain-things@0.1.0` packages the Xiaohei-style article illustration skill for pi, including references, example images, and OpenAI skill metadata.

Use it for article/blog inline illustrations, shot lists, image edits, and visual metaphors for workflows, structures, states, and key ideas. It defaults response text and handwritten image labels to English unless another language is explicitly requested.

Install:

```bash
pi install npm:@diegopetrucci/pi-extensions
# or standalone:
pi install npm:@diegopetrucci/pi-illustrations-to-explain-things
```

Then run `/reload`.
